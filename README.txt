# Finance Tracker

A lightweight, browser-based personal finance tracker that allows users to manage income, expenses, categories, 
and generate visual reports. Data is stored locally in the browser using `localStorage`, making it easy to use 
without a backend server.

---

## Features

- **User Accounts**
  - Register and login securely.
  - Each userâ€™s data is private and stored separately.
  
- **Income & Expenses**
  - Add, edit, and view income and expense entries.
  - Assign transactions to categories.
  
- **Categories**
  - Default categories included (Salary, Bonus, Investment, Other).
  - Add custom categories for better tracking.
  
- **Dashboard**
  - Shows Current Balance, Total Income, Total Expenses, and Savings Rate.
  - Hover effects on KPI cards for better interactivity.
  
- **Reports**
  - Dynamic charts for Income vs Expenses and Expenses by Category.
  - Recent transactions displayed in a table.
  - Charts update in real-time per user.

- **Settings**
  - View user profile (Name & Email).
  - Toggle Dark/Light mode.
  
- **Responsive Design**
  - Fully responsive layout for desktop, tablet, and mobile.
  
- **LocalStorage-Based**
  - All data stored locally in the browser; no backend required.
  
---

## Tech Stack

- **HTML5 / CSS3 / JavaScript**
- **Chart.js** for interactive charts
- LocalStorage for storing user and transaction data
- Custom CSS for layout and theme
